# Roadmaps

This section is now intentionally thin.

Most of the old roadmap material has been removed from the public docs tree.
What remains here is only a minimal landing page and a small number of
repository artifacts that are not part of the rendered documentation site.

## What still lives here

- `docs/08-roadmaps/README.md`: this landing page
- `docs/08-roadmaps/gbif-challenge-2026.html`: standalone presentation kept in
  the repository, but not published in the rendered docs site

## Why this section changed

The old roadmap folder had become a mix of target architecture, execution
plans, GUI rollout notes, and temporary analysis documents. Keeping all of that
as a public top-level section made the docs harder to trust and harder to
navigate, so the surviving public docs now point readers back to current
architecture in [../07-architecture/README.md](../07-architecture/README.md)
instead of treating old planning material as active reference content.
