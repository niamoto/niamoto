# External APIs Reference

Niamoto talks to external services in several parts of the product rather than
through a single central gateway.

## Main integration areas

- Taxonomy and spatial enrichment loaders under
  `src/niamoto/core/plugins/loaders/`
- Deployment integrations under `src/niamoto/core/plugins/deployers/`
- Desktop preview and GUI backend routes under `src/niamoto/gui/api/`
