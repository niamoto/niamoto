# CLI Commands Reference

The CLI itself is the source of truth for flags and subcommands. This page
points to the main command groups and the docs that explain how they fit
together.

## Main command groups

The current command groups live under `src/niamoto/cli/commands/`.

Common entry points:

- `niamoto init`
- `niamoto import`
- `niamoto transform`
- `niamoto export`
- `niamoto deploy`
- `niamoto run`
- `niamoto gui`
- `niamoto plugins`
- `niamoto stats`
- `niamoto optimize`

## Check the installed version

For the exact options supported by your installed version:

```bash
uv run niamoto --help
uv run niamoto <command> --help
```
